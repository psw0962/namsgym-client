## 사용기술 및 아키텍쳐

javascript, react, nextjs, recoil, styled-components

## 주요 기능

1. 센터 및 트레이너 정보 확인
2. 카카오맵 API를 활용하여 센터 위치 확인 및 검색지로부터 센터 위치 확인
3. 탭 및 센터, 트레이너 이름을 활용한 조건부 리뷰 검색
